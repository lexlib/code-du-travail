# Article L222-1

Les fêtes légales ci-après désignées sont des jours fériés :

- le 1er janvier ;

- le lundi de Pâques ;

- le 1er mai ;

- l'Ascension ;

- le lundi de Pentecôte ;

- le 14 juillet ;

- l'Assomption ;

- la Toussaint ;

- le 11 novembre ;

- le jour de Noël.
